# Frame By Frame (ИС для фотостудии)

# Функционал ИС:
   - авторизация;
   - CRUD-операции;
   - формирование отчетов в MS Excel, Word;
   - отображение статистики на графике;
   - расписание занятости фотографа на календаре.
